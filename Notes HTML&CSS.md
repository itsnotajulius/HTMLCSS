Lists:

| Description      |     | Tags   |
| ---------------- | --- | ------ |
| Unordered List   |     | `<ul>` |
| Ordered List     |     | `<ol>` |
| List Item        |     | `<li>` |
| Description List |     | `<dl>` |
| A2               | B2  | C2     |
| A3               | B3  | C3     |

- Unordered HTML List

```html
<ul>
  <li>Coffee</li>
  ...
</ul>
```

- Ordered HTML List

```html
<ol>
  <li>Coffee</li>
  ...
</ol>
```

- HTML Description Lists

```html
<dl>
  <dt>Coffee</dt>
  <dd>- black hot drink</dd>
  ...
</dl>
```
